## Information Diet (Summary)

* There is enough established knowledge. No need to track events live.
* If something matters, it will be analysed and persist.

**Rule:**
If I don’t need to act, I can wait.

**Avoid (default):**

* live feeds, hot takes, constant updates
* reactive use of tools (including ChatGPT)

**Prefer:**

* long-form, expert analysis, durable sources

**Trust model:**

* don’t seek perfect trust
* assign weight, compare over time

**Engagement:**

* wait for write-up + critique before engaging
* if it doesn’t reach that stage, ignore

**Structure:**

* deep work
* real rest (no scrolling)
* 1–2 input windows (RSS first, optional bounded social)

**Principle:**
Be deliberate, not reactive.
