Remove stale SSH host key for a host	ssh-keygen -R <host>
What does `ssh-keygen -R <host>` remove?	The old host key entry from `known_hosts`
After removing a host key, next SSH connection does what?	Prompts you to accept the new host key
SSH option to accept new host keys automatically	`StrictHostKeyChecking=accept-new`
Ansible setting that disables SSH host key checking	`host_key_checking = False`
